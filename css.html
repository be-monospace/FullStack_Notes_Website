CSS - Cascading Style Sheets

É uma linguagem de folhas de estilos utilizadas para descrever a apresentação visual de um documento HTML.

Versões:

  • CSS 1 - Dezembrp 1996
  • CSS 2 - Maio 1998
  • CSS 2.1 - Junho 2011
  • CSS 3 - primeiro módulo publicado em Março 2013
  • CSS 4 - não existe nenhuma especificação única e integrada com a designação CSS 4, mas já existem vários módulos que se encontram na versão 4 (ex: Selectors Level 4) e outros que vão entrar na versão 5 (ex: Media Queries Level 5).

Ao contrário das versões 1, 2 e 2.1 que foram versões que definiam várias funcionalidades num único "documento", a partir das CSS3 começou-se a trabalhar por módulos que foram sendo disponibilizados à medida que ficavam prontos. Isso significa que actualmente ainda se estão a publicar módulos.

Só um número pequeno de módulos das CSS3 está terminado (fase Recommendation). A maior parte dos módulos está na fase Candidate Recommendation, Proposed Recommendation ou Working Draft (que significa que as propostas ainda estão a ser trabalhadas).

O trabalho efectuado em cada um dos módulos pode ser visto na seguinte página: https://www.w3.org/Style/CSS/current-work.en.html
